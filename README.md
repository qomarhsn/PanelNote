## Overview
**PanelNote Extended** is a lightweight extension designed for short reminders or quick notes.

> This is a maintained fork of [PanelNote](https://github.com/GittyMac/PanelNote) by [@GittyMac](https://github.com/GittyMac), updated to implement note positioning in the panel with extended support for GNOME Shell version 49.

## Features
* **Edit Easily:** Click the panel note and type in the popup menu.
* **Auto-Save:** Text is automatically saved as you type.
* **Custom Positioning:** Configure in settings to align Left, Center, or Right on the panel.
* **Precise Placement:** Set the exact position index to control the order relative to other icons.
* **Native Look:** Blends seamlessly with GNOME Shell.
